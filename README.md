# Profile Card Component

## Descripción

Profile card UI component built with HTML and CSS, focused on spacing, typography and reusable visual structure.

## Tecnologías

- HTML
- CSS

## Objetivo

Construir un componente visual reutilizable practicando spacing, tipografía y estructura.

## Funcionalidades

- Tarjeta de perfil
- Jerarquía visual
- Estilos reutilizables
- Componente compacto

## Aprendizajes

Mejora de criterio visual, espaciado y estructura CSS para componentes UI.

## Mejoras futuras

Crear variantes, añadir responsive y publicar demo.
